## Requirements
- Android Studio
- Firebase
- Git

# Installation
- Clone this repository in your preferred directory

` git clone https://github.com/ssinghal13/Oops-Project.git `
- Install Android Studio

` follow the link https://developer.android.com/studio/install `
- Open the cloned project in Android Studio
- Set up your own Firebase Project and connect with your Android Project

` follow the link https://firebase.google.com/docs/android/setup `
- After the setup is done, you can build and run the project in simulator/android device.
